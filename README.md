# what-does-discord-translate-for-this

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts <lang-code> <search term>
```

where search term can be discord's intl hash, the key, or just a string in english

This project was created using `bun init` in bun v1.3.6. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
